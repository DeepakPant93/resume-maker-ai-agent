1. Initialize the repository if it's your first time:

    ```bash
    cd resume-maker-ai-agent
    make init-repo
    ```

2. Install dependencies using Poetry:

    ```bash
    make bake-env
    ```

3. Run the FastAPI server:

    ```bash
    make run
    ```
